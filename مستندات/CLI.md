# عقد CLI (Takween CLI Contract)

## الأوامر الرسمية (MVP)
- `takween تهيئة`
- `takween init`
- `takween بناء`
- `takween build`
- `takween تشغيل`
- `takween run`
- `takween تنظيف`
- `takween clean`

## السلوك المتوقع

### `تهيئة / init`
- ينشئ قالب مشروع مبدئي:
  - `مشروع.تكوين`
  - `المصدر/`
  - `المصدر/الرئيسية.baa`

### `بناء / build`
- يقرأ `مشروع.تكوين`.
- يفشل مباشرة عند أي خرق للمواصفة.
- في MVP المبكر: قد يقتصر على التحقق وإعلان readiness.

### `تشغيل / run`
- يستدعي build أولاً.
- عند نجاح build: يشغّل الناتج.

### `تنظيف / clean`
- يحذف مخرجات البناء فقط (مثل `بناء/`).

## أكواد الخروج (Exit Codes)
- `0`: نجاح.
- `1`: خطأ تنفيذي عام.
- `2`: خطأ استخدام CLI (أمر/وسائط غير صحيحة).
- `3`: أمر معروف لكنه غير مفعل بالكامل بعد.

## رسائل المساعدة
عند عدم تمرير أمر أو تمرير أمر غير معروف، يجب عرض سطر استخدام واضح يتضمن الأسماء العربية والإنجليزية.
