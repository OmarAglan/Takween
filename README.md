# تكوين (Takween) 🏗️

> نظام البناء الرسمي وإدارة الحزم للغة البرمجة **باء (Baa)**.
> *The official build system and package manager for the Baa programming language.*

**تكوين** هو البديل العربي والحديث لأنظمة البناء المعقدة مثل `CMake` أو `Make`. تمت كتابته بالكامل بلغة باء، وهو مصمم ليكون بسيطاً، سريعاً، ويعتمد على مبدأ "الاصطلاح قبل التهيئة" (Convention over Configuration).

## ✨ الميزات (Features)
- **بسيط جداً:** لا حاجة لتعلم لغة برمجة جديدة لكتابة سكربتات البناء. ملفات التكوين تستخدم صيغة `مفتاح: قيمة` واضحة.
- **مستقل بالكامل:** مكتوب بلغة باء، ولا يعتمد على بايثون، C++، أو أي أدوات خارجية (باستثناء مترجم باء نفسه).
- **بناء ذكي (قريباً):** يعيد تجميع الملفات التي تم تعديلها فقط (Incremental Builds).
- **إدارة الحزم (قريباً):** جلب المكتبات الخارجية وربطها بمشروعك تلقائياً.

## 📂 هيكل المشروع القياسي (Standard Project Structure)
عند استخدام تكوين، يفترض النظام الهيكل التالي لمشروعك:

```text
مشروعي/
├── مشروع.تكوين      <-- ملف إعدادات البناء
├── المصدر/          <-- مجلد الكود المصدري (.baa)
│   └── الرئيسية.baa
└── بناء/            <-- المجلد التلقائي للمخرجات (.exe / .o)
```

## 📝 مثال لملف `مشروع.تكوين`
```text
الاسم: تطبيقي_الأول
الإصدار: ١.٠.٠
النوع: تنفيذي
التحسين: ٢
المصدر: المصدر/
المخرج: بناء/
```

## 🚀 الاستخدام (Usage)
*(ملاحظة: تكوين قيد التطوير حالياً)*
```powershell
# لتهيئة مشروع جديد
takween تهيئة

# لبناء المشروع
takween بناء

# لبناء وتشغيل المشروع
takween تشغيل

# لتنظيف مخرجات البناء
takween تنظيف
```
