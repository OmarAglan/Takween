- **آخر تحديث:** 2026-03-02
- **مرجع Baa:** `v0.4.4.1`
- **النطاق الحالي:** Windows-first + Installer عبر Inno Setup + PATH-ready UX.

## قرارات تصميم مثبتة
- **لغة الأوامر:** العربية أولاً + aliases إنجليزية.
- **النطاق المنصّي:** Windows-first في MVP.
- **Strictness:** parser صارم (Fail-Fast).
- **اكتشاف الملفات في MVP:** `ملفات:` صريحة.
- **التوزيع:** Inno Setup + تثبيت مستخدم بسيط + PATH.

## المخاطر الحالية والتخفيف
- **اختلاف بيئات المستخدمين (PATH/صلاحيات):**
  - التخفيف: مثبت يعمل بصلاحيات مستخدم + إعداد PATH للمستخدم الحالي.
- **عدم توفر Baa على الجهاز:**
  - التخفيف: رسالة تنبيه بعد التثبيت مع خطوات بسيطة.
- **مشاكل toolchain مع مسارات غير ASCII:**
  - التخفيف: build script داخلي يعتمد staging ASCII للبناء.

## ما بعد MVP
- دعم Linux installer/package.
- بناء ذكي (Incremental + DAG).
- مدير حزم.
- Dogfooding لبناء أجزاء من Baa عبر Takween.
